A Few Projects

Below are some projects that I have completed during my time at Lambda.

Responsive Design Web Page

In this project, the html was provided and I designed the page by writing CSS. My CSS demonstrates the following:

  • Write CSS so that my page matches designs that were provided.
    Links to design files: desktop, tablet, and mobile.
  • The use of the CSS Flexbox Layout module.
  • Using max-width media queries to adjust website for different devices.

Team Built Landing Page

I worked with a partner to design a landing page for the fictitious AtoZ Experiences company. My contribution was the navigation bar, footer, Safety page, and the Contact page. This project demonstrates:

  • The design and build a navigation bar.
  • The use of responsive units for text.
  • My ability to collaborate to make a single product.
  • The use of the CSS Flexbox Layout module.
  • Using max-width media queries to adjust website for different devices.

JavaScript Fundamentals

In this project, we applied our knowledge of functions, arrays, and objects. This project demonstrates:

  • The use of callback functions.
  • The creation of objects and object methods.
  • The use of forEach, map, filter, and reduce methods.
  • The creation of contructor functions, prototypes, and the 'this' keyword.
  • Converting prototypes into classes.

DOM, Components, and Event Listeners

Here I used JavaScript to manipulate the DOM, create components, and wrote event listeners for the page. This project demonstrates:

  • Basic DOM manipulation.
  • Selecting and designing elements using JavaScript.
  • Creating DOM components with Javascript functions.
  • The use of events and event listeners.
  • *Added animations for menu and article buttons.

React-UI-Components - Lambda Calculator

In this project, I created a React App calculator by importing data and passing it down to components. This project demonstrates:

  • Importing data to state.
  • Passing down portions of this data to different components.
  • Different components mapping over their data and creating different calculator buttons
  • Importing different componets into App.js and styling them to match design file.
  • *Using JavaScript to make calculator functional.

React: Component Side Effects

This project features the use of React to retrieve data from an API and then display it using components. This project demonstrates:

  • The use of the NASA API to fetch the "Astronomy Photo Of The Day"
  • Adding an effect hook to handle the API call side effect.
  • Placing the called data in state and creating components to display it on the website.
  • *Creating a component that can make API calls from different dates.

*React Router Apple Nav

I created a react application that mimics Apple's navigation bar and sub-navigation bars. This project demonstrates:

  • The use of BrowserRouter, Routes, and NavLinks.
  • Creating a navigation bar from imported data.
  • Passing data down to different submenu components
  • Submenu components calling in another component that maps through data and creates submenus.

React To Do List App

A React application that uses React classes to take data from an input field and set it to state. Also added the ability to remove selected items from state. This project demonstrates:

  • An input field that allows user to add items to state.
  • Ability to change state of list items between completed and not completed by clicking.
  • A clear field button that updates list by removing completed items.
  • *Search field that filters through list and displays matching results.
  • *Button that restores To Do list after performing search.

* Extra work that was not required.
